# 🚗 MoveVehicle 🚓

MoveVehicle is a FivePD plugin that enhances traffic stops by allowing the stopped vehicle to move to the next safe space on the roadside. This helps to keep traffic stops safe and orderly by ensuring that stopped vehicles are not obstructing traffic. 🚦

## ✨ Features

- 🚗 Moves the stopped vehicle to the next safe roadside location
- 🛑 Clears all tasks from the driver before moving
- 📢 Provides feedback and notifications to the player

## 📥 Installation

### Quick & Simple 😇
1. **Download**: the latest `MoveVehicle.net.dll` from releases.
   
2. **Install**: the plugin into your FivePD `plugins` folder.

3. **Restart**: FivePD and the plugin should work!
   
### DIY 🔨
1. **Download the Plugin**: Clone or download the repository from GitHub.

    ```sh
    git clone https://github.com/DrDankus/MoveVehicle.git
    ```

2. **Build the Plugin**: Open the project in your preferred IDE (e.g., Visual Studio) and build the project to create the DLL file.

3. **Install the Plugin**:
    - Place the compiled `MoveVehicle.dll` file into the `plugins` folder of your FivePD installation.

4. **Configure FivePD**:
    - Ensure that FivePD is properly installed and configured on your server.

## 🚀 Usage

- **Initiating a Move**:
    - During a traffic stop, press the `Q` key (default) to move the stopped vehicle to the next safe roadside location.
